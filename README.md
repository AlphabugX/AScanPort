# AScanPort
AScanPort 高速、多线程、全端口、单IP扫描。

食用方法

```
  -d string
    	text|json| (default "text")
  -o string
    	result.txt (default "result_20220415_172833.txt")
  -t string
    	127.0.0.1 or f5.ink (default "f5.ink")
```
